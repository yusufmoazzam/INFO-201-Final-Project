Part I: Find the Data

Dataset 1: State Crime Rates

Data Source: https://corgis-edu.github.io/corgis/csv/state_crime/

Data Meaning: This data represents information on crime rates and total crimes across the United States of America from 1960 to 2019. Crimes have been split into two main categories: property and violent crimes.

Data Join: We intend to join using state and year. A feature we need to change from the state column in this dataset is that the states start with an uppercase letter. We must use the toLower() function since joining is case-sensitive. Another issue is the years. Not only are the early years naturally less accurate data, the other dataset only starts at 2013. Therefore, we will make the year range 2013-2019.

Data Observations: There are 3,115 rows and 21 columns. The relevant column names with example values are state (“Washington”), Year (2015), Population (2105856), Rates.Property.All (23632), Rates.Violent.All (204.6), Totals.Property.All (45246)), and Totals.Violent.All(15537).

Data Issues: The only potential cause of unreliability is the accuracy of the early years of crime data, which we mitigate by deleting all rows before 2013

Dataset 2: FBI Uniform Crime Reporting (UCR) Program Data 2013-2019

Data Source: https://www.openicpsr.org/openicpsr/project/117974/version/V3/view?path=/openicpsr/117974/fcr:versions/V3/human_trafficking_2013_2019_csv.zip&type=file

Data Meaning: This is an FBI data set that is a part of the annual Uniform Crime Reporting. It contains information about human trafficking offenses in the United States. There are two main crime categories: Commercial sex acts and involuntary servitude.

Data Join: We intend to join using state and year. This dataset will need no changes and the other dataset will conform to its year range and state casing.

Data Observations: There are 162,635 rows and 37 columns. The relevant column names with example values are state (“washington”), year (2013), population (2105856), reported.commercial.sex.acts (2), reported.involuntary.serv (4) reported.total (6).

Data Issues: The main problem with this data set is the fact that human trafficking is one of the most unreported crimes. The data is limited to what has been reported/what they have on file which means there are countless cases unaccounted for.

Part III: Data Documentation

##Codebook ##State Crime Rates dataset: We downloaded this CSV file from GitHub, where it was used in another project. But the original dataset comes from the CORGIS project, which gathers real crime data from U.S. government sources. The information on the dataset can be found on this website, which explains the columns and variables: https://corgisedu.github.io/corgis/csv/state_crime/

##FBI Human Trafficking dataset: This describes how the FBI’s Uniform Crime Reporting (UCR) data is collected and explains the variables that represent human trafficking (reported, actual, unfounded, and cleared counts). Information about the dataset is provided on the OpenICPSR page: https://www.openicpsr.org/openicpsr/project/117974/version/V3/view?path=/openicpsr/117974/fcr:versions/V3/human_trafficking_2013_2019_csv.zip&type=file

Most Relevant Variables

  • state: used to join datasets and compare by location
  • Year: used to join datasets and analyze trends
  • Data.Population: helps tell about the size of each state
  • Data.Rates.Property.All: overall property crime rates
  • reported_total: number of reported cases
  • actual_total: number of comfirmed cases These variables are important becasue the allow us to compare overall crime reports and population levels with human trafficking reports.
